# Research

This directory contains supporting research materials, literature reviews, and background information.

## Purpose

Use this space to:
- Document existing solutions and alternatives
- Compile relevant academic papers or articles
- Summarize market research
- Track competitive analysis
- Store data and statistics

## Recommended Structure

```
research/
├── README.md (this file)
├── literature-review.md
├── competitive-analysis.md
├── market-research.md
├── technical-feasibility.md
├── user-research.md
└── sources/
    └── [PDFs, links, notes]
```

## Literature Review Template

See [literature-review.md](literature-review.md) for the standard format.

## Citation Format

When citing sources, use a consistent format:

**Academic Papers:**
> Author(s). (Year). Title. *Journal*, Volume(Issue), pages. DOI

**Web Articles:**
> Author. (Year, Month Day). Title. *Website*. URL

**Books:**
> Author. (Year). *Title*. Publisher.

## Organization Tips

- **Tag sources** by theme or relevance
- **Summarize key findings** - don't just link
- **Track publication dates** - note if sources are outdated
- **Note credibility** - distinguish between peer-reviewed and opinion pieces
