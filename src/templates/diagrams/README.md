# Diagrams

This directory contains visual documentation for your idea.

## Recommended Diagram Types

### Architecture Diagrams
- System architecture
- Component relationships
- Data flow

### User Experience
- User flows
- Wireframes
- Mockups
- Prototypes

### Process Diagrams
- Workflows
- State machines
- Sequence diagrams

## Tools

Consider using these tools for creating diagrams:
- **Mermaid** - Text-based diagrams (can embed in markdown)
- **Excalidraw** - Hand-drawn style diagrams
- **Figma** - UI/UX design
- **draw.io** - General-purpose diagramming
- **PlantUML** - UML diagrams from code

## Mermaid Example

You can embed Mermaid diagrams directly in markdown:

\`\`\`mermaid
graph LR
    A[User] --> B[Frontend]
    B --> C[API]
    C --> D[Database]
\`\`\`

## File Organization

Organize your diagrams by type:
- `architecture/` - System architecture
- `ux/` - User experience designs
- `flow/` - Process flows

## Export Formats

- PNG/SVG for images
- Mermaid/PlantUML for text-based diagrams
- Figma links for interactive prototypes
