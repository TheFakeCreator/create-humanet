# Document Templates

This directory contains reusable templates for common documentation needs.

## Available Templates

- **decision-record.md** - Architecture Decision Record (ADR) template
- **meeting-notes.md** - Meeting notes template
- **research-summary.md** - Research summary template

## Usage

1. **Copy** the template to the appropriate directory
2. **Rename** with proper naming convention
3. **Fill in** the sections
4. **Delete** unused sections if not applicable

## Creating New Templates

If you create new templates that might be useful for others:

1. Add them to this directory
2. Document the purpose and usage
3. Use clear section headers
4. Include examples where helpful

## Template Guidelines

Good templates should:
- Have a clear structure
- Include instructions in [brackets]
- Show examples of good content
- Be comprehensive but not overwhelming
- Use consistent formatting

---

*Templates help maintain consistency across documentation.*
